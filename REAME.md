### Install
- Clone repo
- Install dependencies with yarn or npm
```bash
yarn install || npm i
```

### Run 
- Set process variables
- Run server with `npm run start` or `yarn start`

### Variables
| name                      | type   | default |
|---------------------------|--------|---------|
| STATIC_PORT               | Number | 8000    |
| API_PORT                  | Number | 8001    |
| MONGODB_CONNECTION_STRING | String | -       |
